# Recipio Api

## Init project

```
npm init -y
```
```
npm i typescript -D
```

```
npx tsc --init
```

## Useful libs

tsx to run typescript code locally
```
npm i tsx -D
```

tsup to build typescript code to javascript code
```
npm i tsup -D
```

vitest for testing, has better support for typescript
```
npm i vitest -D
```
